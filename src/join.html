how do I join LinkBush?

you'll need to be a member of Freakybob Team and in the Freakybob Dev Chat 3 server.

To create an account, make a thread in the #get-linkbush channel and label it like this:

Username wanted (aka URL): [username wanted, ex: wish]

Accounts: [your account/website URLs]

Pronouns: [your pronouns]

⚠️ LinkBush is in beta and currently does not support styling. In order to style your LinkBush, you'll need to edit it yourself.

how do I edit my LinkBush?

you'll have to use the Discord thread you made to sign up. send a new message with the info you'd like edited, and bam, we'll do it!

rules

you must let us add your linkbush URL to our directory.
